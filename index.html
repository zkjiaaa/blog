第一篇 github 博客

博客说明

今天忙里偷闲，看到了一篇文章教我们怎么使用 hexo 搭建自己的博客。非常感兴趣。决定自己搭建一个。

刚开始是想搭建在 gitee 上面，因为国内访问比较快嘛。但是没想到居然需要过手持身份证，觉得比较麻烦便决定在 github 上搭建吧。

说一下本博客会分享的东西：

  • 技术文档：在日常工作中遇到的觉得比较耗时才解决的问题会在这里进行复盘
  • 基金/股票： 在投资中的一些感悟和见解也会在这发表（不代表投资建议）
  • 生活感悟： 人生无常，突然想写点什么就会在这里记录一下
